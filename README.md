# RH Analytics - Análise de Turnover com SQL e Power BI

## 📌 Objetivo

Este projeto foi desenvolvido com o objetivo de analisar os fatores que influenciam o desligamento de colaboradores e auxiliar a área de Recursos Humanos na tomada de decisões para redução do turnover.

---

## 🎯 Problema de Negócio

A empresa apresentava um elevado índice de desligamentos e precisava responder à seguinte pergunta:

> Quais fatores estão influenciando o desligamento dos colaboradores e onde devemos agir para reduzir o turnover?

---

## 🛠 Ferramentas Utilizadas

- SQL
- Power BI
- Excel (apoio exploratório)

---

## 📂 Estrutura do Projeto

### Etapa 1 - SQL

#### Nível 1 - Exploração de Dados

- Total de funcionários
- Funcionários desligados
- Taxa de turnover
- Quantidade de departamentos
- Quantidade de cargos
- Salário médio
- Idade média
- Tempo médio de empresa

**Conceitos utilizados**

- COUNT()
- SUM()
- AVG()
- CASE WHEN
- ROUND()

---

### Nível 2 - Agregações

Análises de:

- Turnover por departamento
- Turnover por cargo
- Turnover por gênero
- Turnover por estado civil
- Turnover por faixa etária
- Turnover por escolaridade
- Salário médio por departamento
- Salário médio por cargo
- Tempo médio de empresa por departamento
- Média de idade por cargo

**Conceitos utilizados**

- GROUP BY
- HAVING
- ORDER BY
- CASE WHEN

---

### Nível 3 - Perguntas de Negócio

- Quais cargos possuem maior índice de desligamento?
- Funcionários que fazem hora extra saem mais?
- Existe relação entre satisfação e desligamento?
- Existe relação entre salário e turnover?
- Qual departamento possui maior custo salarial?
- Funcionários solteiros saem mais?
- Viagens frequentes impactam o turnover?
- Work-Life Balance influencia o turnover?
- Quais departamentos possuem funcionários mais antigos?
- Ranking de cargos por salário médio.

---

### Nível 4 - SQL Avançado

Utilização de:

- CTE
- Window Functions
- OVER()
- PARTITION BY
- RANK()
- DENSE_RANK()
- NTILE()

Consultas desenvolvidas:

- Ranking salarial
- Salário acima da média
- Idade acima da média
- Diferença salarial por departamento
- Percentual de turnover
- Percentual acumulado
- Top 20% cargos mais bem remunerados
- Top 20% departamentos com maior turnover
